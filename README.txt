Wonkasoft Logout is a simple plugin for the adding of a redirect.

== Description ==

Wonkasoft Logout was created for the purpose of quickly and easily to give specific direction or redirection to wordpress
for when someone logs out of or logs into their account.

== Installation ==

1. Enter plugins from the admin menu.
1. Click Add New then search for Wonkasoft Logout then click Install Now.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Follow the settings link under the plugin name on the Plugins page.
1. Or you can find the settings page for Wonkasoft Logout under the Settings tab in the admin menu.

== Frequently Asked Questions ==

= Can I add a custom link for redirecting =

Yes, We have as of version 1.1.0 you are able to add a custom url for your redirecting on login and logout.
